---
title: terrateam apply-autoapprove
description: Execute a plan and apply in the same execution, skipping approvals
---

To execute a plan and apply in the same execution, skipping approvals, use the following command:
```
terrateam apply-autoapprove <optional-tag-query>
```
[Learn more about our tag queries](/advanced-workflows/tags).

:::caution
This is a dangerous operation. See [access control](/configuration-reference/access-control) to limit access.
:::
